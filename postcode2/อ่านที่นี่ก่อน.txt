ทำ Dropdown ให้เลือกจังหวัด อำเภอ ตำบล และทำหน้าจอแก้ไข ให้มีการกำหนดค่าเริ่มต้น

*วิธีการติดตั้ง*

1. ให้ ไปที่ Control Panel ของเว็บไซต์ของคุณ สร้างฐานข้อมูลชื่อ thai_province (จะเป็นชื่ออื่นก็ได้) ก่อน
2. พอคลาย ZIP ออกมา จะพบไฟล์ Sourcecode
3. เข้าไปแก้ไขข้อมูลการเชื่อมต่อฐานข้อมูล ในไฟล์ที่ชื่อ config.php (ตั้งให้สอดคล้องกับที่สร้างไว้ในข้อ 1. ) และเมื่อแก้เสร็จแล้วให้กด Save ด้วย
4. ให้ Upload ไฟล์ เข้าไปบน Server ของเว็บไซต์
5. ให้เข้าไปยัง phpMyAdmin ของเว็บไซต์ของคุณ (ไปดูที่ Control Panel ของ Host คุณ มันจะมี Link ไปอยู่)
6. ให้เลือกฐานข้อมูลที่คุณได้สร้างไว้ในข้อ 1.
7. ให้เลือกไปที่ Import -> กด Browse แล้วเลือกไฟล์ thai_province.sql เสร็จแล้วกดปุ่ม Go (ให้เลือก Character set เป็น utf-8 ด้วย)
8. ถ้านำเข้าข้อมูลสำเร็จ จะเห็นรายการ จังหวัด อำเภอ ตำบล ในตาราง province , amphur , district , geography 
9. เป็นอันเสร็จสิ้นขั้นตอนการติดตั้ง ระบบสามารถใช้งานได้เลย โดยเรียกใช้งานได้ทาง http://www.sitename.com/ajaxprovincecitytambon/example.html
